# Guides for Researchers

These guides are for **using jsPsych to build and run experiments** — no plugin development required.

**New to jsPsych?** Start with the [Hello World tutorial](../../getting-started/hello-world.md) to get an experiment running in about five minutes, then work through the [reaction-time tutorial](./tutorials/rt-task.md) to learn the core ideas — timelines, plugins, and data.

**Looking for a specific capability** — randomization, dynamic parameters, data handling, surveys, eye-tracking? Browse the **Feature Overview**.
