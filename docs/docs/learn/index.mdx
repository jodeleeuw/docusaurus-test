# Learn jsPsych

Everything you need to design and run behavioral experiments with jsPsych. If you also want to *build* plugins, extensions, or timelines, head to [Extend jsPsych](../extend/index.mdx).

**New to jsPsych?** Start with the [Hello World tutorial](../getting-started/hello-world.md) to get an experiment running in about five minutes, then work through the [reaction-time task tutorial](./tutorials/rt-task.md) to learn the core ideas.

How this section is organized:

- **Tutorials** — hands-on, start-to-finish walkthroughs.
- **Core concepts** — the building blocks of every experiment: timelines, plugins, extensions, dynamic parameters, and the data object.
- **Guides** — task-focused how-tos, grouped by what you're trying to do.
- **Deploy & recruit** — host your experiment and collect data from participants online.
